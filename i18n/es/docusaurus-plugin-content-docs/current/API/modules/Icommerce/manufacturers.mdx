# Fabricante

Usa la REST API para interacturar con los fabricantes en la aplicación.

:::info
Recuerda que para algunas peticiones requieres de enviar el token de autenticación.

`Header Parameters`
```json
{
  "Authorization": "Bearer ...token..."
}
```
Y los parametros de consulta son los siguientes:

`Query Parameters`
```json
{
  "setting": {
    "timezone":"America/Bogota",
    "fromAdmin":true,
    "appMode":"iadmin",
    "authProvider":"local",
    "locale":"es"
  }
}
```
:::

##  Modificar un Fabricante

Usa este endpoint para modificar una fabricante. Puedes enviar uno o varios valores en el request.

Por defecto el valor que se acepta es el id, pero si quieres enviar otro valor, tienes que enviar en los `Query Parameters`:

```json
{
  "filter": {
    "field": "title" //Valor por el que se va a pasar la búsqueda
  }
}
```

<Api
  method="PUT"
  url="/api/icommerce/v3/manufacturers/{criteria}"
  minimum={`{
  "attributes": {
    "es": {
      "title": "change name",
    }
  }
}`}
  request={`{
  "attributes": {
    "store_id": 1,
    "status": "1",
    "sort_order": "0",
    "medias_single": {},
    "medias_multi": {},
    "es": {
      "name": "test",
      "slug": "test",
      "description": "Test",
      "meta_title": "",
      "meta_description": ""
    }
  }
}`}
  responseSample={`{
  "data": "Item Updated"
}`}
/>

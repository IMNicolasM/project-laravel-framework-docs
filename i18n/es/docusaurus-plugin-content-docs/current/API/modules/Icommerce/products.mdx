# Productos

Usa la REST API para interacturar con los productos en la aplicación.

:::info
Recuerda que para algunas peticiones requieres de enviar el token de autenticación.

`Header Parameters`
```json
{
  "Authorization": "Bearer ...token..."
}
```
Y los parametros de consulta son los siguientes:

`Query Parameters`
```json
{
  "setting": {
    "timezone":"America/Bogota",
    "fromAdmin":true,
    "appMode":"iadmin",
    "authProvider":"local",
    "locale":"es"
  }
}
```
:::

##  Modificar un producto

Usa este endpoint para modificar un producto. Puedes enviar uno o varios valores en el request.

Por defecto el valor que se acepta es el id, pero si quieres enviar otro valor, tienes que enviar en los `Query Parameters`:

```json
{
  "filter": {
    "field": "name" //Valor por el que se va a pasar la búsqueda
  }
}
```

##  Obtener varios productos

Usa este endpoint para obtener varios producto.

Puedes filtrar los productos, tambien paginarlos e incluir datos (Como si trae las categorias asociadas, o no)
Los datos que se pueden incluir son:

| Include | Descripción |
|------------|--------------------------------------------|
| categories | Trae las categorias asociadas al producto. |
| manufacturer | Trae los datos del fabricante, asociado al producto. |
| tags | Trae las etiquetas asociadas al producto. |

<Api

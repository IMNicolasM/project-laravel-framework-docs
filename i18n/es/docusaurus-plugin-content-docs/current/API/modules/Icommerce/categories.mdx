# Categoría
Usa la REST API para interacturar con las categorías en la aplicación.

:::info
Recuerda que para algunas peticiones requieres de enviar el token de autenticación.

`Header Parameters`
```json
{
  "Authorization": "Bearer ...token..."
}
```
Y los parametros de consulta son los siguientes:

`Query Parameters`
```json
{
  "setting": {
    "timezone":"America/Bogota",
    "fromAdmin":true,
    "appMode":"iadmin",
    "authProvider":"local",
    "locale":"es"
  }
}
```
:::

##  Modificar un Categoría

Usa este endpoint para modificar un categoría. Puedes enviar uno o varios valores en el request.

Por defecto el valor que se acepta es el id, pero si quieres enviar otro valor, tienes que enviar en los `Query Parameters`:

```json
{
  "filter": {
    "field": "title" //Valor por el que se va a pasar la búsqueda
  }
}
```

<Api
  method="PUT"
  url="/api/icommerce/v3/categories/{criteria}"
  minimum={`{
  "attributes": {
    "es": {
      "title": "fix name",
    }
  }
}`}
  request={`{
  "attributes": {
    "user_id": 2,
    "status": "1",
    "parent_id": "0",
    "show_menu": false,
    "featured": "0",
    "sort_order": "0",
    "medias_single": {},
    "medias_multi": {},
    "layout_id": null,
    "es": {
      "title": "categorías",
      "slug": "categorias",
      "description": "testeans",
      "h1_title": "",
      "meta_title": "",
      "meta_description": ""
    }
  }
}`}
  responseSample={`{
  "data": "Item Updated"
}`}
/>

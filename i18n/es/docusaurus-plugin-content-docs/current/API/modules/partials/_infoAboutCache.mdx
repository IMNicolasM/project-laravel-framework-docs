<summary>Do you want to view your products?</summary>

:::info

Después de utilizar la API para acceder a los productos, es importante actualizar la caché para visualizar los cambios.

¿Cómo hacerlo?

- Dirígete a la esquina superior derecha y selecciona el menú desplegable con tu nombre de usuario.
- Haz clic en 'Configuraciones'.
- En la sección 'Acciones', busca la opción 'Limpiar caché'.
- Haz clic en 'Limpiar caché' para actualizar la información y visualizar los productos actualizados.

:::
